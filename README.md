# Emergency Portal BD

A premium, production-ready emergency services portal for Bangladesh, built with React (Vite), Tailwind CSS, Framer Motion, and Lucide icons.

## Getting started

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## What's inside

- **Hero** with a signature "dispatch console" panel showing the national 999 hotline
- **Live stats, emergency categories, and quick-dial numbers** with call/copy actions
- **Live alerts timeline**, **first aid knowledge base**, **hospital finder**, **blood donor search with filters**, **disaster shelter grid**, **interactive map placeholder**
- **Report Emergency** form with validation, loading and success states (react-hook-form)
- **FAQ accordion**, **testimonials**, and a full **footer**
- Fully responsive (320px → 1920px), keyboard accessible, reduced-motion aware

## Stack

React 19 (Vite), React Router DOM, Tailwind CSS, Framer Motion, Lucide React, React Hook Form, React Hot Toast.
